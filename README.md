## Introduction

The AI Resume Enhancer is a powerful tool designed to help job seekers optimize their resumes for specific job applications. By leveraging advanced AI models from Anthropic and Google, this application analyzes resumes against job descriptions and company information, providing detailed feedback and suggestions for improvement.

## Features

- Resume analysis based on job description and company information
- AI-powered resume optimization
- Detailed feedback and suggestions for improvement
- Score comparison between original and optimized resumes
- Support for multiple AI providers (Anthropic's Claude and Google's Gemini)
- File upload (PDF, TXT) and text input options
- Downloadable JSON report with analysis and optimization details
- Downloadable optimized resume
- Quantitative scoring comparisons to track improvements
